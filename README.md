## How to run the project
1. Clone the repo.

2. Make sure to have nodejs installed:

        https://nodejs.org/en/

3. Run the following command:

    ````
    npm install
    ````

4. When all the dependencies is installed run this command to start the program:

    ````
    npm run start
    ````

5. Go to http://127.0.0.1:5050/
